# FastAPI Test Tasks

Подборка выполненных тестовых заданий на FastAPI для различных вакансий и стажировок.

## 🎯 Цель

Продемонстрировать практические навыки разработки REST API с использованием FastAPI: проектирование эндпоинтов, работа с базами данных, асинхронная обработка запросов и написание тестов.

## 🗂️ Проекты

Каждый проект находится в отдельной ветке этого репозитория и содержит собственный `README.md` с подробной инструкцией по запуску.

- **Lesta Games** — Парсер и веб-сервис для обработки данных ([перейти к проекту](https://github.com/Zheka-m-p/FastAPI_test_tasks/tree/lesta_games))
- **AVB Invest (5D HUB)** — Cервис для сокращения URL с асинхронной обработкой ([перейти к проекту](https://github.com/Zheka-m-p/FastAPI_test_tasks/tree/avbinvest))
